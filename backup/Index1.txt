<h2>LiteGappsX</h2>

<img class="responsive-img MID" style="width: 60%;" src="/img/s2.webp" alt="litegapps 2">

<p>LiteGappsX is a development of regular Litegapps which is able to support various versions of Android and multiple architectures with just one <b>flashable.zip</b> while still maintaining a small size.  but we do not provide variants in LitegappsX.</p>
<p>We aim this for those who often change ROMs with different Android versions and only need one flashable ZIP.</p>

<p>LiteGappsX only has the same core Google Apps components as <b>litegapps variant lite</b>, so it focuses on minimalism. Litegapps++ is divided into three types, namely:</p>
<h3>Reguler</h3>
<p>LiteGappsX
Regular is updated regularly using the latest version of Playstore and Google Play Services, but it only supports Android 9 (Pie) to Android 14.</p>
<h3>MicroG</h3>
<p>LiteGappsX MicroG uses MicroG as the default gms, it has extensive support for Android 5.0 (lollipop) up to the latest Android 14 and is very light.  But many of the main functionalities don't work, such as: payments, connecting accounts to games and many others.</p>

<h2>Addon</h2>
<p>LiteGapps Addon is an additional package that we provide to suit user needs, users can download which package they like and then use it, to complete the LiteGapps package that doesn't have a complete package.</p>
<p>It also supports installation via <i><b>Magisk module, KSU module, recovery, adb sideload, litegapps controller</b></i> and you can even do so without installing the main litegapps package.</p>

<h2>LiteGapps Remover</h2>
<p>Litegapps Remover is a tool for debloating Google Apps on Android Roms, to overcome Google Apps that are too many and heavy on the default Rom.</p>

</p>Supports systemless and non-systemless installations, it is recommended to install LiteGapps Remover before installing Litegapps for ROMs that already include Google Apps</p>

<h3>Systemless</h3>
<p>Litegapps Remover supports installation as a Magisk module or KSU module, which means you can debloat it systemlessly and restore the debloated application completely.</p>

<h3>Non systemless</h3>

<p>Litegapps remover can also be installed as recovery (non systemless), this will be permanently deleted on the partition.</p>

<h2>LiteGapps Controller</h2>

<div><img class="responsive-img MID" style="width: 60%;" src="/img/s3.webp" alt="litegapps 3"></div>
<p>LiteGapps controller is an additional feature of LiteGapps and LiteGapps++, which has many features and tweaks for the user experience in using LiteGapps on Android devices.</p>

<p>For complete instructions on how to use it, you can read the documentation and below are some of the LiteGapps controller features:</p>

<b>Usage</b>
<p>Open Terminal Emulator or termux</p>

<p><code>su</code></p>

<p><code>litegapps</code></p>


<h3>Boost</h3>
<p>This boosts your device, by forcing applications to stop in the background, this will also make CPU and GPU performance lighter due to the load from applications that are not in use.</p>
<h3>Gaming Mode</h3>
<p>Gaming mode functions to optimize the user experience when playing games, this will deactivate applications that are not used when playing games to make more efficient use of RAM memory and battery, and has three modes, namely:</p>
<ul>
<li>
<b>Gaming Mode 1</b> This mode will force the application to stop in the scaled list every 60 seconds it is running.
</li>
<li>
<b>Gaming Mode 2</b> This mode will disable the applications in the list, deactivate them if you want to restore the application.
</li>
<li>
<b>Gaming Mode 3</b> This mode will deactivate applications that are on the user's desired list.
</li>
</ul>
<h3>Battery Saver</h3>
<p>Save battery by force stopping some applications in the background and reducing CPU and GPU performance.</p>
<h3>Device Info</h3>
<p>information on your device's software and hardware.</p>
